# FlashcardMSB_Visual
Projeto: FlashcardMSB.css  Descrição: O FlashcardMSB.css é um projeto desenvolvido para criar uma interface moderna e interativa de flashcards educacionais. Seu principal objetivo é auxiliar estudantes no processo de memorização e revisão de conteúdos, utilizando um design limpo, responsivo e fácil de personalizar.
